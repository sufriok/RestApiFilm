# RestApiFilm

CRUD Rest API

Laravel 5.8 <br>
PHP 7.2.8 <br>
passport:7.5.1 <br>

copy ".env.example", rename to ".env" (setting database) <br>
composer install <br>
php artisan key:generate <br>
php artisan migrate <br>
php artisan db:seed <br>
php artisan passport:install <br>
php artisan serve <br>
